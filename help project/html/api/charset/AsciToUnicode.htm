功能
将 Ansi 字符串转换为 Unicode 字符串。

参数
ZStrPtr：
Ansi 字符串指针。

ZStrLen：
字符串数据长度，传递 0 则自动计算字符串长度。

返回值
返回转换后的字符串，如果转换失败，返回 NULL。

补充说明
使用此函数转换字符集，返回的字符串指针不用时需要释放，否则会造成内存泄漏。
