功能
将 Unicode 字符串转换为 Utf-8 字符串。

参数
UStr：
Unicode 字符串指针。

ULen：
字符串数据长度，传递 0 则自动计算字符串长度。

返回值
返回转换后的字符串，如果转换失败，返回 NULL。

补充说明
使用此函数进行字符集转换不需要释放内存，引擎内部会负责内存的回收使用。
