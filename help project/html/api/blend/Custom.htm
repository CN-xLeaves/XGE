功能
Blend渲染器核心函数。

参数
src：
源图像指针。

px：
绘制到目标的横坐标。

py：
绘制到目标的纵坐标。

cx：
源图像裁剪起点横坐标。

cy：
源图像裁剪起点纵坐标。

cw：
源图像裁剪宽度。

ch：
源图像裁剪高度。

dst：
目标图像指针。

bk：
混合处理函数回调。

param：
传递给混合处理函数的额外参数。

返回值
成功返回 -1[True] 失败返回 0[False]。

补充说明
本函数负责将两个图像表面进行自定义算法混合。
函数具备完善的边界检查功能，会自动对越界的部分进行裁剪和跳出。
