功能
直接改变文件大小。

参数
file：
文件路径。

size：
新的文件大小。

返回值
成功返回 -1[True] 失败返回 0[False]。

补充说明
大部分编程语言都有自带的文件读写函数封装，文件系列函数仅提供一种更简单的方式访问文件。
新的文件大小如果超出原文件大小，则多出的数据会用 \0 覆盖，如果比原文件小，则多的那部分会被删除。
