功能
打开文件。

参数
file：
文件路径。

readonly：
是否以只读方式打开。

返回值
成功返回文件句柄，失败返回 NULL。

补充说明
大部分编程语言都有自带的文件读写函数封装，文件系列函数仅提供一种更简单的方式访问文件。
此函数相当于 CreateFile ，可用API函数或使用 XGE 提供的 Put_File、Get_File、File_Len 进行操作。
