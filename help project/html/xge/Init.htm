功能
初始化XGE游戏引擎。

参数
b：
游戏场景颜色深度，可传递 32 或 16，分别对应32位深度和16位深度(65535色)。

w：
游戏场景宽度(非全屏模式可使用任意数值，全屏模式下则需要使用屏幕支持的特定分辨率)。

h：
游戏场景高度(非全屏模式可使用任意数值，全屏模式下则需要使用屏幕支持的特定分辨率)。

flag：
可以指定窗口初始化的一组标记，使用 or[按位或运算] 或 加法 运算连接，可参考：数据参考.引擎初始化。

lockfps：
锁定游戏刷新率，指定40则每秒刷新40次，指定为 0 则不锁定刷新率。

title：
创建游戏窗口的标题，传递 NULL 使用默认标题。

返回值
成功返回 -1[True] 失败返回 0[False]。

补充说明
使用其他 xge 函数前，都必须使用这条命令先初始化。
这条命令可以多次执行，一次性初始化的内容不会再次初始化，可用于改变分辨率。

使用范例
C++：
// 使用32位色，800x600窗口化，每秒画面刷新40次不使用垂直同步默认标题初始化
xge::Init(32, 800, 600, xge_window, 40, false, NULL);

FreeBasic：
// 使用16位色，1024x768全屏模式，开启垂直同步
xge.Init(16, 1024, 768, xge_fullscreen, 0, -1, "我的游戏")
