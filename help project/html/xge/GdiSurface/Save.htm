功能
保存图像到硬盘。

参数
tpe：
要保存的文件类型，目前这个参数是无效的，仅支持保存为bmp格式图像。

f：
保存图像的文件路径。

q：
当保存格式为jpg时，这个参数指定了压缩率，因为目前还无法保存为其他格式，这个参数是无效的。

返回值
无返回值。

补充说明
参数为未来升级预留，目前仅支持保存为bmp图像，未来可能会支持多种图像格式输出。
