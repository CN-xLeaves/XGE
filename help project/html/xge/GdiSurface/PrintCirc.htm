功能
在图像中画圆形或填充圆形。

参数
x：
圆形起点横坐标。

y：
圆形起点纵坐标。

w：
圆形宽度。

h：
圆形高度。

c：
圆形线条或填充颜色。

返回值
无返回值。

补充说明
与 xge.shape 中的圆形有所区别，这里的圆形是在矩形范围内绘制的，所以参数 x、y 不是起点坐标。
