功能
在图像中画线。

参数
x1：
线条起点横坐标。

y1：
线条起点纵坐标。

x2：
线条终点横坐标。

y2：
线条终点纵坐标。

c：
线条颜色。

返回值
无返回值。

补充说明
x1、y1、x2、y2组成两个点，线在两点之间绘制。
