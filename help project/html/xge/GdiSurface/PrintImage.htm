功能
在图像中贴图。

参数
x：
贴图起点横坐标。

y：
贴图起点纵坐标。

f：
贴图文件路径或内存地址，受 LoadMode 设置载入模式的影响。

w：
贴图宽度 [可用于图像缩放]。

h：
贴图高度 [可用于图像缩放]。

cx：
裁剪起点横坐标。

cy：
裁剪起点纵坐标。

cw：
裁剪宽度。

ch：
裁剪高度。

返回值
无返回值。

补充说明
PrintImage [普通贴图，按dpi计算原图大小]
PrintImage2 [普通贴图，按像素计算原图大小]
PrintImageZoom [可缩放贴图]
PrintImageEx [可缩放可裁剪贴图]
PrintImageFull [画刷矩形填充贴图]
