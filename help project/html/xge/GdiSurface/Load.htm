功能
释放旧的图像，并从文件中载入图像。

参数
f：
要加载图像的文件路径，如果从内存中加载图像，则为内存指针。

返回值
成功返回 -1[True] 失败返回 0[False]。

补充说明
LoadMode 设置为 0[False] 的情况下，本函数从文件中加载图像，否则从内存中加载图像。
使用构造函数加载图像时指定的加载方式也会更改这里的设定。
