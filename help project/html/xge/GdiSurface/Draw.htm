功能
将图像渲染到画面或另一幅图像中。

参数
x：
将图像渲染到目标的横坐标。

y：
将图像渲染到目标的纵坐标。

cx：
裁剪图像的起始横坐标。

cy：
裁剪图像的起始纵坐标。

cw：
裁剪图像的宽度。

ch：
裁剪图像的高度。

sf：
将图像渲染到的目标图像，传递 NULL 则将图像渲染到场景中。

返回值
无返回值。

补充说明
Draw [使用透明色渲染(紫红色&HFF00FF为XGE的固定透明色)]
Draw_Alpha [使用图像自带的透明通道渲染，仅支持32位，16位色深按Draw方式画出]

带有 Ex 的渲染模式在原有基础上允许对原图片进行裁剪，使用cx、cy、cw、ch四个参数确定裁剪的范围。
