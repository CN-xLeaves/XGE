功能
根据颜色边界，填充指定范围。

参数
x：
填充点横坐标。

y：
填充点纵坐标。

c：
填充色。

p：
填充纹理，传递一组纹理数据，可使用纹理创建工具创建。

f：
进行填充时判断的边界颜色。

sf：
要绘制到的图像指针，如果传递NULL，则绘制到场景中。

返回值
无返回值。
