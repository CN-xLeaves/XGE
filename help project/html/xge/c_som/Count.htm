功能
获取当前管理器中已使用的音频总数[音频释放后也算已使用]。

参数
无参数。

返回值
返回当前管理器中已使用的音频总数。

补充说明
这个函数获取的数据相当于计数器数据，音频释放后数据会被清空，但计数器不变，依然算一个。
当音频管理器预申请的音频已经满了，会自动寻找之前释放过的音频。
