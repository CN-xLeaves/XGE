功能
创建一幅图像，并添加到管理器中。

参数
可参考：
xge::Surface.Create。
xge::Surface.Load。

返回值
返回图像在管理器中的 idx，在管理器中使用图像，都需要通过这个 idx 进行操作，失败返回 0。

补充说明
请妥善保管这个 idx ，对于管理器而言，他代表着一副图像。
