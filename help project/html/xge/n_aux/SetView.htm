功能
设置游戏场景视图。

参数
x1：
视图起点横坐标。

y1：
视图起点纵坐标。

x2：
视图终点横坐标。

y2：
视图终点纵坐标。

c：
清除可视区域的颜色。

f：
是否在可视区域四周画线条围起来。

返回值
无返回值。

补充说明
并不算很常用的功能，不理解可以跳过，用于剪辑游戏画面。
