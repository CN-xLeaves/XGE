功能
为场景指定新的坐标系范围。

参数
x1：
新坐标系起点横坐标。

y1：
新坐标系起点纵坐标。

x2：
新坐标系终点横坐标。

y2：
新坐标系终点纵坐标。

c：
是否使用第一象限表示纵坐标。

返回值
无返回值。

补充说明
可使用新的坐标系来改变整个场景的绘制位置，制造震动效果，或实现其他的作用。
