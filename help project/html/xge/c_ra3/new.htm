FreeBasic：
Constructor()

Constructor(f As ZString Ptr)
Constructor(w As Integer, h As Integer)
Constructor(dp As Any Ptr, ds As Integer, dw As Integer, dh As Integer, db As Integer)

SDK：
SDK版本不支持构造特性。

功能
渲染对象创建时初始化图像及其他参数。

参数
sf：
直接使用现有的Surface时传入此参数。

f：
从文件创建Surface并设置为渲染对象图像时传入此参数。

w、h：
创建一个指定大小的空 Surface 并设置为渲染对象图像时传入此参数。

dp、ds、dw、dh、db：
从内存中载入 bitmap 数据创建 Surface 并设置为渲染对象图像时传入此参数。

补充说明
第一种构造方式是空构造，初始化时不执行任何操作。
第二种构造方式使用现有的 Surface 作为渲染对象的图像，渲染对象释放时不会释放 Surface。
第三种构造方式在渲染对象创建时载入一张图片作为渲染图像，渲染对象释放时会将这幅图像一起释放。
第四种构造方式在渲染对象创建时创建一幅空白图像作为渲染图像，渲染对象释放时会将这幅图像一起释放。
第五种构造方式在渲染对象创建时载入一幅内存图像作为渲染图像，渲染对象释放时会将这幅图像一起释放。
创建图像相关参数解释请参考：xge::Surface.构造。
