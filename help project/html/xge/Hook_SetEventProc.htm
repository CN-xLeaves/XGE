功能
设置当前使用的消息分发器函数地址。

参数
n：
新的消息分发器函数地址。

返回值
无返回值。

补充说明
设置新的消息分发器，会使旧的消息分发器失效。
除非要破坏原有的消息分发体系，否则尽量备份旧的消息分发器函数。
也可以在新的消息分发器执行后调用旧的消息分发器，实现多分发器共存。
默认的消息分发器被替换后，游戏场景将不能再收到大部分的设备输入消息。
