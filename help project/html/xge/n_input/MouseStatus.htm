功能
获取鼠标的各项状态。

参数
x：
用于接收鼠标横坐标位置的整数指针。

y：
用于接收鼠标纵坐标位置的整数指针。

w：
用于接收鼠标滚轮计数的整数指针。

b：
用于接收鼠标按钮状态的整数指针。

返回值
无返回值。

补充说明
鼠标按钮状态是一组位标记，使用 and 运算 1、2、4 可分别获取 左键、右键、中键 是否按下。
在FreeBasic中，因为input存在命名冲突，因此命名空间应使用input_x。
