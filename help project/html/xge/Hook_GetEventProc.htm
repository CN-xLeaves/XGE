功能
获取当前使用的消息分发器函数地址。

参数
无参数。

返回值
返回当前使用的消息分发器函数地址。

补充说明
消息分发器处理游戏场景对设备输入及系统消息的分发处理，可用于对引擎进行二次开发和功能扩展。
配合 Hook_SetEventProc 函数修改消息分发器，能拦截游戏的输入消息。
