功能
在图像或场景的指定位置写字。

参数
x：
写字起点横坐标。

y：
写字起点纵坐标。

w：
写字宽度格式化范围。

h：
写字高度格式化范围。

f：
字体名字。

px：
字体大小。

flag：
字体特征和格式化显示方式，可参考：数据参考.文字风格。

c：
文字颜色。

txt：
文字内容。

weight：
描边边框宽度。

c1：
文字颜色。

c2：
描边颜色。

fi：
纹理文件路径或内存地址，受 LoadMode 设置载入模式的影响。

返回值
无返回值。
