功能
计算点是否在圆形范围内。

参数
px：
点的横坐标。

py：
点的纵坐标。

cx：
圆心的横坐标。

cy：
圆心的纵坐标。

cl：
圆的半径。

返回值
在圆形范围内返回 -1[True] 否则返回 0[False]。
