功能
计算点是否在矩形范围内。

参数
px：
点的横坐标。

py：
点的纵坐标。

x1：
矩形左上角的横坐标。

y1：
矩形左上角的纵坐标。

x2：
矩形右下角的横坐标。

y2：
矩形右下角的纵坐标。

返回值
在矩形范围内返回 -1[True] 否则返回 0[False]。
