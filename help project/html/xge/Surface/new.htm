定义

 C++：
Surface();
Surface(int w, int h);
Surface(char* f);
Surface(void* p, int s, int w, int h, int b);

FreeBasic：
Constructor()
Constructor(w As Integer,h As Integer)
Constructor(f As ZString Ptr)
Constructor(p As Any Ptr, s As Integer, w As Integer, h As Integer, b As Integer)

SDK：
SDK版本不支持构造特性。

功能
指定一种类初始化时图像加载/创建方式。

参数
w：
要加载或创建的图像宽度。

h：
要加载或创建的图像高度。

f：
从文件加载图像时指定图像路径。

p：
从内存加载图像数据时指定图像数据的内存指针。

s：
从内存加载图像数据时指定图像数据的大小。

b：
从内存加载图像数据时指定图像数据的位深。

补充说明
第一种构造方式是空构造，初始化时不执行任何操作。
第二种构造方式会创建一副新图像，图像使用透明色填充。
第三种构造方式会从文件中加载一副图像，按图像文件的大小创建图像。
第四种构造方式从内存中加载BitMap数据，需要指定色深和图像大小。
更多信息请参照：Create、Load。
