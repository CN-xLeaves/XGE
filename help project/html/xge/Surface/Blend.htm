功能
将图像渲染到画面或另一幅图像中，使用自定义混合算法渲染。

参数
x：
将图像渲染到目标的横坐标。

y：
将图像渲染到目标的纵坐标。

cx：
裁剪图像的起始横坐标。

cy：
裁剪图像的起始纵坐标。

cw：
裁剪图像的宽度。

ch：
裁剪图像的高度。

bk：
自定义混合算法函数的指针。

param：
自定义混合算法函数的附加参数。

sf：
将图像渲染到的目标图像，传递 NULL 则将图像渲染到场景中。

返回值
无返回值。

补充说明
此方式相比 Draw_Custom 方式更快，但较难掌握，本质是直接的内存块运算。
