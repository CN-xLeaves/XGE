功能
从文件中读取数据。

参数
file：
要读取的文件路径。

buffer：
用于存储读取数据的内存指针。

addr：
读取文件的开始位置。

lenght：
读取数据长度。

返回值
成功返回 -1[True] 失败返回 0[False]。

补充说明
大部分编程语言都有自带的文件读写函数封装，文件系列函数仅提供一种更简单的方式访问文件。
