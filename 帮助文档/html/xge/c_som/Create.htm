功能
创建一个音频，并添加到管理器中。

参数
可参考：
xge::Sound.Create。
xge::Sound.Load。

返回值
返回音频在管理器中的 idx，在管理器中使用音频，都需要通过这个 idx 进行操作，失败返回 0。

补充说明
请妥善保管这个 idx ，对于管理器而言，他代表着一个音频。
