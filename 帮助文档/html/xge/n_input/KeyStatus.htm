功能
获取指定按键的状态。

参数
k：
按键扫描码，可参考：数据参考.按键扫描码。

返回值
返回指定按键的状态。

补充说明
正常情况下，XGE游戏引擎会在场景消息中通知按键消息，此函数仅作备用。
在FreeBasic中，因为input存在命名冲突，因此命名空间应使用input_x。
