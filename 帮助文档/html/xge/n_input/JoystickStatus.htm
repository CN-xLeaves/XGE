功能
获取游戏手柄的各项状态。

参数
id：
设备编号，可选值 0-15。

btn：
用于接收按钮状态的整数指针。

a1：
用于接收第一个方向数值的小数指针。

a2：
用于接收第二个方向数值的小数指针。

a3：
用于接收第三个方向数值的小数指针。

a4：
用于接收第四个方向数值的小数指针。

a5：
用于接收第五个方向数值的小数指针。

a6：
用于接收第六个方向数值的小数指针。

a7：
用于接收第七个方向数值的小数指针。

a8：
用于接收第八个方向数值的小数指针。

返回值
成功返回 0[False] 失败返回 1[True]。

补充说明
按钮状态的存储方式与 MouseStatus 获取的方式类似，每一位代表一颗按钮的状态。
方向数值的范围在 -1.0 与 1.0 之间，如果不存在方向控制器，则返回 -1000.00。
在FreeBasic中，因为input存在命名冲突，因此命名空间应使用input_x。
